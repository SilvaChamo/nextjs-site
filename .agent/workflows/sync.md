---
description: Sincronizar o projeto com o GitHub (Add, Commit, Push)
---

Este workflow automatiza a sincronização do projeto:

1. Adiciona todas as mudanças:
// turbo
`git add .`

2. Cria um commit com as últimas alterações:
// turbo
`git commit -m "Actualização automática"`

3. Envia para o repositório remoto:
// turbo
`git push origin main`
